# **Proyecto de Cuentas Bancarias🏦**
Este proyecto lo realicé para el curso Python Intermedio de Tecno3F.
Simula un sistema bancario básico utilizando POO en Python. Se implementaron clases para representar diferentes tipos de cuentas bancarias, aplicando conceptos como Encapsulamiento, Herencia, Clases abstractas y Polimorfismo.

## **Objetivos del Proyecto🎯**
* Aplicar POO en Python.
* Utilizar clases abstractas y métodos abstractos.
* Implementar herencia.
* Encapsulamiento de atributos y métodos.
* Simular operaciones bancarias básicas: depósito, extracción y cálculo de intereses.
* Cálculo de la edad del titular de la cuenta.

## **Funcionamiento del programa💻**
1. El usuario elige el tipo de cuenta que desea utilizar: ahorro o corriente
2. Se solicitan los datos del titular: Nombre, DNI y Fecha de nacimiento.
3. Se ingresa el saldo inicial de la cuenta.
4. Se muestra el saldo actual y la edad del titular.
5. El usuario puede realizar una extracción con validaciones según el tipo de cuenta.
6. Se muestra el saldo final.
7. Para cuentas de ahorro, se calcula y muestra el interés generado.
